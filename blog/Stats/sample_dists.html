Distributions of Samples

Let \(Z_i\), \(i = 1, \dots, n\) be iid. standard normal distributed \(Z_i \sim \N(0, 1)\), then \[\sum_{i=1}^n Z_i^2 \sim \chi^2_n.\]

For \(U_1 \sim \chi^2_{k_1}\), \(U_2 \sim \chi^2_{k_2}\) independent \[\frac{U_1/k_1}{U_2/k_2} \sim F_{k_1,k_2}.\]

For \(T \sim t_n\) \[T^2 \sim F_{1,n}, \quad T^{-2} \sim F_{n,1}.\]

Let \(X_i\), \(i = 1, \dots, n\) be iid. normal distributed \(X_i \sim \N(\mu, \sigma)\), then \[ \overline X = \frac{1}{n}\sum_{i=1}^n X_i \sim \N(\mu, \frac{\sigma}{\sqrt{n}}), \] \[ \sum_{i=1}^n (X_i - \overline X)^2 \sim \sigma^2 \chi^2_{n-1} \quad \text{(is independent from } \overline X), \] \[ n(\overline X - \mu)^2 \sim \sigma^2 \chi^2_1, \] \[ \left(\frac{\overline X - \mu}{\hat \sigma /\sqrt{n}}\right)^2 = \frac{n(\overline X - \mu)^2}{\frac{1}{n-1} \sum_{i=1}^n (X_i - \overline X)^2} \sim \frac{\chi^2_1}{\frac{1}{n-1}\chi_{n-1}^2} \sim F_{1,n-1} \] \[ \frac{(n-1) \hat \sigma^2}{\sigma^2} \sim \chi^2_{n-1}, \] \[ \frac{\overline X - \mu}{\hat \sigma /\sqrt{n}} \sim t_{n-1} \] See Cochran's theorem.
